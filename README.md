# CCNA Network Labs

Three Packet Tracer labs built while preparing for the CCNA 200-301, 
demonstrating routing, switching, security, and NAT concepts through 
hands-on configuration, verification, and troubleshooting.

## Labs

### [Lab 1: OSPF Redundant Core with VLAN Segmentation and DHCP](./lab1-ospf-core)
Multi-router OSPF core with automatic failover, VLAN-segmented 
departments, and DHCP relay across subnets.

### [Lab 2: Router-on-a-Stick with ACL Segmentation and Port Security](./lab2-roas-security)
Inter-VLAN routing via subinterfaces and 802.1Q trunking, with an ACL 
enforcing department-level access restrictions and port security on 
access ports.

### [Lab 3: NAT Overload with Secured Remote Management](./lab3-nat-ssh)
NAT overload (PAT) connecting an internal network to a simulated 
internet, with SSH-only remote management restricted by ACL.

## Skills Demonstrated
OSPF | VLANs | Trunking (802.1Q) | DHCP | ACLs | Port Security | NAT/PAT 
| SSH | Static Routing | VLSM | Troubleshooting
